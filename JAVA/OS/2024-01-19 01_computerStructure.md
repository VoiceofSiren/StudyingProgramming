# Computer Structure
<br/>

### 이 글의 목적
    - 동일한 프로그래밍 언어와 로직으로 개발해도 일부 시스템에서는 동작하지 않는 경우가 있는데,
      이때 컴퓨터 구조에 대해 분석해야 하는 경우가 있다.
    - 또한 성능, 용량, 비용을 고려한 개발을 위하여 컴퓨터 구조에 대한 이해는 필수적이다.
<br/>

### 1. 컴퓨터가 이해하는 정보
#### 1) 데이터
    - 숫자, 문자 등의 정적인 데이터
#### 2) 명령어
    - 컴퓨터를 실직적으로 움직이게 하는 정보
