# ES6
<br/>

### JavaScript ES6
    - JavaScript ES6는 ECMAScript 6의 약자.
    - 2015년에 발표된 JavaScript 최신 표준.
<br/>

### 1. 변수 선언 방식의 개선
- var, let, const 키워드를 사용하여 변수를 선언할 수 있다.
#### 1) var
    - 전역 변수 선언
    - scope: 함수
#### 2) let
    - 지역 변수 선언
    - scope: {} 블록
#### 3) const
    - 지역 변수 선언
    - 값 변경 불가
<br/>

### 2. Template Literal (템플릿 리터럴)
- 문자열 안에 변수나 표현식을 포함할 수 있다.
#### [코드 1] - Template Literal의 예시
```plaintext
  const name = "John Doe";
  const message = `Hello, ${name}!`;
```
#### [결과 1]
```plaintext
  Hello, John Doe!
```
