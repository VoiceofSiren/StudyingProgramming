# EC2
<br/>

### 이 글의 목적
    - EC2의 Route 53에 대해 알아보고자 한다.
<br/>

### 1. Route 53
    - Domain을 등록하거나 DNS를 구성하는 DNS 서비스이다.
- 서비스 종류
    - EC2 Instance
    - S3 Bucket
    - Load Balancer
<br/>
